# I7 Laravel 12 CRUD Reference Project

This repository serves as a comprehensive reference project demonstrating the standard coding patterns, helpers, and CRUD implementations used across I7 projects (ERP, Booking, Bike, etc.).

It is built on **Laravel 12** and includes a complete suite of developer documentation and live examples.

## Features Included

1. **I7 Helpers Integration**
   - `helperForm`: AJAX form submission, validation error handling, form reset.
   - `helperSwal` & `helperConfirm`: SweetAlert2 wrappers for consistent dialogs.
   - `helperInputs`: Auto-initialization of input masks (decimal, integer, email, date) and pickers (Flatpickr, Select2).
   - `helperJS` & `helperUI`: General utilities and UI blocking during AJAX calls.

2. **PHP Helpers**
   - `ApiHelper`: Standardized `sendResponse()` for all AJAX endpoints.
   - `FilesHelper`: Consistent `uploadImage()`, `getImageUrl()`, and `deleteFile()`.
   - `MainHelper`: Date formatting and general utilities.

3. **CRUD Examples**
   - **Simple CRUD (Modal-based)**: Create, edit, and delete records using Bootstrap modals loaded via AJAX. Includes image upload and inline status toggles.
   - **Advanced CRUD (Page-based)**: Complex forms with multiple columns, image previews, password toggles, and multi-delete functionality using checkboxes.
   - **CRUD with Sort**: Drag-and-drop sorting using jQuery UI Sortable, saving the order via AJAX.

4. **Developer Documentation**
   - Built-in documentation pages explaining how to use the helpers, DataTables, AJAX patterns, input components, and layout structure.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd crud-laravel
   ```

2. Install dependencies:
   ```bash
   composer install
   npm install
   ```

3. Environment setup:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. Configure your database in `.env`, then run migrations and seeders:
   ```bash
   php artisan migrate --seed
   ```

5. Link storage for images:
   ```bash
   php artisan storage:link
   ```

6. Serve the application:
   ```bash
   php artisan serve
   ```

## Project Structure

- `app/Helpers/`: Contains all PHP helper functions.
- `app/Http/Controllers/Developer/`: Contains the controllers for the CRUD examples and documentation.
- `resources/views/layouts/cpanel/`: The main layout, sidebar, header, and inline JS helpers.
- `resources/views/developer/`: Blade views for the CRUDs and docs.
- `public/modules/developer/js/`: Page-specific JavaScript files for DataTables and form handling.

## Coding Standards

- **AJAX First**: All form submissions and state changes (like status toggles or deletes) are handled via AJAX.
- **Validation**: Laravel FormRequests are used for validation. Errors are automatically caught by `helperForm` and displayed below the respective inputs.
- **DataTables**: Server-side processing using Yajra DataTables.
- **UI Components**: Bootstrap 5, SweetAlert2, Select2, Flatpickr, and Inputmask.
